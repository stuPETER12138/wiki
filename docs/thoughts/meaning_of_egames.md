# 电子游戏的意义

电子游戏于我们有什么意义？是会带来认知提升，锻炼信息处理能力，提高理解速度和决策能力；创造社交连接，强化沟通能力；带来情感上的满足；提供历史与文化的教育。还是说电子游戏会造成注意力低下；造成抑郁，从而与社会脱节；产生病态的精神依赖；忽略对于现实的追求？

由此探讨下去，我们的讨论必将莫名地向游戏究竟是好还是坏的方向发展。

那么，以最为广泛的视角，结合电子游戏的本质来看：当我们问道”电子游戏的意义是什么？”，无疑是在问“**想象是否拥有意义？**”

与电影相似的是，电子游戏也是在用一种自身领域所能行驶的独特方式，将一段经由想象编排的故事、事件乃至世界具现出来，摆放到游玩者（观影者）的面前。当我们剥开画面、音效、规则以及商业化外壳后，电子游戏最为核心的的内核是一段着重强调结构化，并可以提供交互的”想象本身“。

于是乎，“为何那么多人热爱游戏”就可以被写作为“为何人们那么热衷于想象”。人类为何需要想象？想象为我们带来了什么？想象到底对人们做了什么？

想象是我们逃离既定世界的出口。“想象”为我们提供了一个几乎不需要付出任何代价的，可以称之为平行于甚至凌驾于这个现实世界的世界。游戏世界从不会将死亡以生命的代价记录在它的世界逻辑之中。死亡从原本泯灭性质，摇身一变成为了再次出发以及勇敢尝试的鼓励。想象为我们带来的是绝对安全的，针对于个体感受的试错机会。我们可以在想象的世界中无限次数的“破产”，无限次数的抒发“冲动”，无限次数的“死去”。即使面对死亡，也要保持足够的耐心与毅力，才能达到自我的目标。想象是人类诞生以来的“意义制造机”，人们会想象未来的工作待遇，会想象在哪些不断机械重复之余的快乐，会想象死亡到来时的自己，会为此焦虑，会为此兴奋，会为此哀悼。

那么，我们便能回答“游戏的意义是什么？”。游戏的意义是一段想象被结构化、被分享、被交互之后的意义。游戏的意义不归任何一个场域所统辖，而是由个体所掌握。游戏的意义是经由人编排制作而出的最为精美的车站，人们从这里乘上幻想的列车，开向另一个能让自己所动容的彼方。人类的想象在这些车站与车站，列车与列车的不断穿梭与会面中产生共振，形成**梦想**。只要人类还在幻想宇宙的模样，还在拒绝既定的人生，想象就永远有意义。

> 我此刻进入的是谁的想象，而我又把自己的想象带进了哪里。
